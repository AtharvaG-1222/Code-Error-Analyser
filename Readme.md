# Error Insight

Lightweight NLP classifier that categorizes common programming errors across multiple languages and provides short explanations.

Supports errors from Python, JavaScript, Java, C++, Go, C#, TypeScript, and general runtime errors.

## Categories

- Syntax Error
- Type Error
- Index Error
- Name Error
- Import Error
- Runtime Error

## Tech stack

- scikit-learn (LinearSVC + TF-IDF)
- Streamlit
- HuggingFace Inference API (optional)

## Setup

```bash
pip install -r requirements.txt
python train.py
streamlit run app.py
```

## Structure

```
├── data/error_messages.csv   # dataset
├── model/                    # saved model (generated by train.py)
├── train.py                  # train classifier
├── predict.py                # predict error type
├── explain.py                # LLM explanation + fallback
├── app.py                    # web UI
└── requirements.txt
```

## LLM explanations

Set `HF_API_TOKEN` for AI-generated explanations via HuggingFace:

```bash
set HF_API_TOKEN=your_token
```

Works without it using built-in fallback text.
